# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release on PyPI | Yes |
| Older releases | No |

## Reporting a Vulnerability

If you discover a security vulnerability in `osint-trust-envelope`, please report it responsibly.

**Do not open a public issue.**

Instead, please use GitHub's
[private vulnerability reporting](https://github.com/WRG-11/osint-trust-envelope/security/advisories/new).

### What to include
- Description of the vulnerability
- Steps to reproduce
- Potential impact (data exposure, information leak, logic bypass)
- Suggested fix or mitigation, if any

### Response timeline
- **Acknowledgment:** within 48 hours
- **Fix (if confirmed):** within 2 weeks
